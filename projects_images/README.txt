Drop project thumbnail images here.

- Use landscape 16:9 images for best results (e.g. 640x360 or larger).
- Reference them in projects.json via the "image" field, e.g. "image": "quest-academy.jpg".
- A filename with no folder is automatically resolved to projects_images/<filename>.
- If "image" is empty or the file is missing, a styled gradient placeholder is shown automatically.
- Optional: add a portrait/tall version and reference it via the "portrait" field.
