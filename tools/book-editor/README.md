# Book editor

## Run it

Double-click `book-editor.cmd` in the website folder. A browser window opens automatically.

Keep the terminal window open while using the editor. Close it or press `Ctrl+C` when finished.

## What it edits

- Book data: `books.json`
- Uploaded covers: `books_images/`
- Automatic backups: `.book-editor-backups/`

The editor runs only on `127.0.0.1`. It does not deploy the website or send book data elsewhere.

## Shortcuts

- `Ctrl+S`: Save changes
- `Ctrl+N`: Create a book

Node.js 18 or newer is required.
