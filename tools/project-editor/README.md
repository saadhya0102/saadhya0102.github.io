# Project editor

## Run it

Double-click `project-editor.cmd` in the website folder. A browser window opens automatically.

Keep the terminal window open while using the editor. Close it or press `Ctrl+C` when finished.

## What it edits

- Project data: `projects.json`
- Uploaded images: `projects_images/`
- Automatic backups: `.project-editor-backups/`

The editor runs only on `127.0.0.1`. It does not deploy the website or send project data elsewhere.

## Shortcuts

- `Ctrl+S`: Save changes
- `Ctrl+N`: Create a project

Node.js 18 or newer is required.
